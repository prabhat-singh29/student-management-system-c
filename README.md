# Student Management System in C

A simple menu-driven Student Management System built using C programming language.

## Features

- Add Student
- Display Students
- Search Student by Roll Number
- Exit Program

## Concepts Used

- Structures
- Arrays
- Loops
- Conditional Statements
- User Input

## How to Run

Compile:

gcc main.c -o main

Run:

.\main
